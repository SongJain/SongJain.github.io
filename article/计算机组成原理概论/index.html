计算机的基本组成

冯 · 诺依曼计算机的特点

1、用二进制表示数据和程序。

2、采取存储程序的方式，按地址寻访。

3、计算机 = 存储器 + 运算器 + 控制器 + I/O设备

4、以 运算器 为中心（这个区别于现代计算机）。

计算机五大部件

运算器：完成算术运算于逻辑运算，并将运算的中间结果暂存在运算器内

存储器：用来存储数据和程序

控制器：控制、指挥程序、数据输入、运行、处理运算结果

输入设备：键盘等

输出设备：显示器，音响等

计算机的五大部件（五大子系统）在控制器的控制下，有条不紊的自动工作。

我们常说的 CPU 就是 运算器 + 控制器 。

计算机工作过程

主存储器（内存或主存）

主存储器包括了三个重要部件：存储体M，MAR寄存器，MDR寄存器。

存储体由许多存储单元组成，存储单元由许多存储元件，存储元件能存储一位二进制数（0或1）。因此一个存储单元能存储一串二进制代码，这一串二进制代码称为存储字。

那么在了解 MAR寄存器和 MDR寄存器之前，我们先要了解一下主存是怎么工作的。主存的工作方式就是按照存储单元的地址号来实现对存储字各位的存（写入）、取（读出），这种方式称为按地址存储方式，简称地址访问存储器（访存）。这种访存的优势在于程序只需要按顺序存储，每次只需要自动加1即可访问下一个指令的地址。

为了实现访存的方式，主存中必须配置两个寄存器。MAR（存储器地址寄存器），用来存储 预访问的存储单元的地址 。MDR（存储器数据寄存器），用来 存放从存储体某个单元中的取出的代码或者准备往存储单元中存入的代码。

运算器

运算器最少包括了三种寄存器和一个算术逻辑单元（ALU）。其中三种寄存器分别为：ACC（累加器）、MQ（乘商寄存器）、X（操作数寄存器）。

加法操作过程：从主存中取出一个存放在M地址号单元内的加数 [M]，送到X寄存器中，然后和被加数[ACC]相加，结果保留在ACC中。即：[M] ->X ; [X] + [ACC] -> ACC。

减法同理加法。

乘法操作过程：从主存中取出存放在M号地址单元的乘数[M]，送到乘商寄存器MQ中，然后再把被乘数送入X寄存器中，并将ACC清0，然后[X]于[MQ]相乘，结果（积）的高位保留在ACC中，低位保留在MQ中。

即：[M] -> MQ ; [ACC] -> X ; 0 -> [ACC] ; [MQ] * [X] -> ACC // MQ

除法同理。

控制器

控制器是计算机的神经中枢。他的运行分成了三个阶段：

1、取指过程。在存储器中读出一条指令。

2、分析过程。对取出的指令进行分析，指出指令完成时什么操作，并且寻址指明操作数的地址。

3、执行过程。根据操作数的地址与指令操作完成操作。

控制器由 PC（程序计数器）、IR（指令寄存器）以及 CU（控制单元）组成。

PC：存放当前准备打算执行程序的地址，他和 MAR 之间有一条直接通道，并且具有自动加一的功能，也就是可以自动走向下一条指令。

IR：用来存放当前的指令，IR 的内容来自主存的 MDR。IR 中的操作码 OP(IR) 传至 CU，用户来分析指令；IR 中的地址 AD(IR) 作为操作数的地址送到存储器的 MAR 。

CU：分析当前指令所需要完成的操作，并发送出微指令，控制所有被控对象。
